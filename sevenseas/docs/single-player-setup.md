# 싱글플레이 우선 개발 기준

## 1) 핵심 게임 루프
1. 항구에서 저가 상품 매수
2. 항해 이동
3. 다른 항구에서 고가 판매
4. 자금으로 선박/장비/인력 강화
5. 더 먼 항로와 고난도 콘텐츠 진입

## 2) 필수 도메인
- 선박: 속도, 내구도, 적재량, 함포 슬롯
- 항구: 위치, 특산품, 시세 계수, 시설
- 경제: 상품 기초가 + 항구 수급 보정
- 플레이어: 자금, 함대 상태, 명성
- 이벤트: 폭풍, 해적, 발견물, 퀘스트

## 3) 온라인 요소 제외 전략
- 거래소/경제는 완전 로컬 계산
- 시간 흐름은 게임 내부 시간으로 고정
- 계정/매칭/실시간 동기화 제거
- 네트워크 의존 API 없이 동작

## 4) 나중에 온라인 전환을 위한 경계
- 상태 저장 인터페이스 분리 (현재 local, 추후 server)
- 경제 계산 모듈 분리 (현재 client, 추후 authoritative server)
- 전투 판정 모듈을 순수 함수 중심으로 유지

## 5) 바로 확장할 우선순위
1. 저장/불러오기
2. 선박 구매/강화 화면
3. 해상 랜덤 이벤트
4. 턴 기반 또는 실시간 전투 프로토타입
5. 지역/국가 평판 시스템
