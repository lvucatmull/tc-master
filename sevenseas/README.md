# Seven Seas (Web + App Ready Prototype)

대항해시대 온라인 감성의 항해/교역 게임을 **싱글플레이**로 구현하는 스타터 프로젝트입니다.
단일 코드베이스로 웹 배포와 앱 패키징을 함께 노릴 수 있게 구성했습니다.

## 요구 사항
- Node.js 20+
- npm 10+

## 빠른 실행
```bash
npm install
npm run dev
```
- 기본 개발 서버: `http://localhost:5173`

## 조작법
- 이동(PC): `← ↑ ↓ →`
- 이동(모바일/터치): 좌하단 터치 컨트롤
- 입항: 항구 근접 후 `E` 또는 `DOCK`
- 교역: 항구 패널에서 `매수/매도`

## 현재 포함된 요소
- 항해: 방향키 + 터치 컨트롤 이동
- 항구: 3개 항구 배치, 근접 시 입항
- 무역: 항구별 시세 차이 기반 매수/매도
- 플레이어 상태: 자금, 화물, 적재량
- 그래픽 톤: 바다 패턴, 파도 레이어, 선박/항구 비주얼 애니메이션
- Wave function 환경: 파고/흐름 함수 기반 바다 라인 렌더링, 선박 흔들림/표류 반영

## 플랫폼 동작 방식
- 웹: 일반 브라우저 실행
- PWA: 설치형 웹앱(홈 화면 추가)
- 앱 래퍼 준비: Tauri/Capacitor에서 동일 빌드 산출물(`dist`) 사용 가능

## 빌드 및 배포
```bash
npm run build
npm run preview
```
- 배포 산출물: `dist/`

## 프로젝트 구조
- `src/game/scenes`: Phaser 씬
- `src/game/systems`: 시세 계산 등 도메인 로직
- `src/game/data`: 항구/상품 데이터
- `src/game/input`: 키보드/터치 통합 입력 상태
- `src/styles`: HUD/패널/모바일 컨트롤 스타일
- `docs`: 설계 및 플랫폼 문서

## 개발 시 알아둘 점
- 현재 저장/불러오기는 미구현(세션 종료 시 진행상황 초기화)
- 그래픽은 코드 기반 프로토타입 텍스처이며 상용 리소스는 포함하지 않음
- 온라인 기능(동기화/매칭/서버 경제)은 의도적으로 제외한 싱글플레이 구조
